ImgArray.grid

Declaration

grid(row,col,...imgarrs)

Function Description

make this array and other ImgArrays into a grid. all array should have the same shape. and the ros*col equals the number of all arrays. this array is first array in the grid.

Parameter Description

  • row: set grid row number.
  • col: set grid col number.
  • imgarrs: arrays to make grid with this array.
